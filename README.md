# Bugün Ne Yapalım?

Kullanıcıya tek dokunuşla rastgele bir günlük aktivite öneren basit ve modern bir mobil uygulama. Uygulama, kararsız kaldığın anlarda seni küçük ama faydalı aksiyonlara yönlendirir.

## Özellikler

- Tek ekranda sade ve modern arayüz
- Rastgele öneri butonu ile anında yeni aktivite fikri
- NativeWind ile responsive tasarım
- Özel Inter fontu ile tutarlı tipografi

## Kullanılan Teknolojiler

- Expo (managed workflow)
- React Native
- TypeScript
- NativeWind
- expo-font
- @expo-google-fonts/inter
